[github.com/qiangli/bashy](https://github.com/qiangli/bashy)

Pure-Go Bash 5.3-compatible shell and agentic command environment for portable developer automation. Ships as a single static binary with Bash-compatible scripting, embedded coreutils, git workflows, dry-run safety checks, and local orchestration commands used across the DHNT toolchain.

**Stack:** Go | Shell runtime | Coreutils | Git | Agentic automation
